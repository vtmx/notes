# kwin

## Shortcuts
```
# Other console
ctrl+alt+f#

# Maximize
meta+up

# Minimize
meta+down

# Show desktop
meta+d

# Switch to next destktop
meta+ctrl+right
meta+ctrl+l

# Switch to previous destktop
meta+ctrl+left
meta+ctrl+h

# Windows to next desktop
meta+shift+right
meta+shift+l

# Windows to previuous desktop
meta+shift+left
meta+shift+h
```
