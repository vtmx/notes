# cmus

Add in library view:

```
:a ~/Music
```

Create playlist:

```
:pl-create <playlist>
```

Add in playlist:

```
:add -p <dir>
```

Order in playlist:

```
p | P = Move from playlist
D     = Remove from play list
```
