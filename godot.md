# Godot

## Definir um nome de classe
```gd
class_name Name
```

## Variáveis
```gd
# Definir tipo da variável
var speed: int = 5

# Exportar variável para a interface
export (int) var speed
```

## Funções nativas
```gd
# Função quando a cena estiver carregada
 _ready():

# Loop comum
_process(delta)

# Loop de física
_physics_process(delta)
```
