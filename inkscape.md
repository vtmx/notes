# Inkscape

## Movimentação

```
Setas         = Move seleção
Alt + Setas   = Move seleção em incrementos menores
Shift + Setas = Move seleção em incrementos maiores
Ctrl + Setas  = Move canvas
```

## Edição

```
Ctrl + D = Clona seleção
```

## Redimensionamento

```
Alt + <>  = Redimensiona seleção
Ctrl + <> = Redimensiona seleção em incrementos maiores
```

## Rotação

```
[]        = Rotaciona seleção
Alt + []  = Rotaciona seleção em incrementos menores
Ctrl + [] = Rotaciona seleção em incrementos maiores
```

## Vira

```
H = Vira seleção horizontalmente
Shift + H = Vira seleção verticalmente
```

## Grupo

```
Ctrl + G = Agrupa seleção
Ctrl + Shift + G = Desagrupa seleção
```

## Mouse Click

```
Click direito = Zoom

ClickM = Movimenta canvas
Shift + ClickM = Seleciona zoom

Scroll = Movimenta canvas verticalmente
Shift + Scroll = Movimenta canvas horizontalmente
Ctrl + Scroll = Diminui e aumenta zoom
```

## Barra de ferramentas

```
V = Seleção
A = Nodes
X = Ferramenta de forma
R = Retângulo
E = Elipse
* = Poligonal
P = Caneta
T = Texto
I = Conta gotas
Shift + P = Páginas
```

## Camadas

```
Ctrl + Shift + N = Nova camada
```

## Paineis

```
Ctrl + Shift + A = Alinhamento e distribuição
Ctrl + Shift + D = Propriedades do documento
Ctrl + E         = Path effects
Ctrl + Shift + E = Exportar
Ctrl + Shift + F = Preenchimento e contorno
Ctrl + Shift + L = Camadas
Ctrl + Shift + N = Nova camada
Ctrl + Shift + M = Transformar
Ctrl + Shift + O = Editor de objetos
Ctrl + Shift + T = Texto e fontes
Ctrl + Shift + Y = Simbolos
Ctrl + Shift + X = Editor XML
Ctrl + Shift + W = Cores
```

## Menu

```
Ctrl + N = Novo
Ctrl + Alt + N = Novo usando template
Ctrl + O = Abrir
Ctrl + S = Salvar
Ctrl + Shift + S = Salvar como
Ctrl + Shift + Alt + S = Salvar cópia
Ctrl + I = Importar
Ctrl + Shift + E = Exportar
Ctrl + P = Imprimir
Ctrl + Shift + D = Propriedades do documento
Ctrl + Q = Fechar
Ctrl + Q = Sair
```

## Exibição

```
Ctrl + 5         = Modo de display
Ctrl + ;         = Guias
Ctrl + Shift + ; = Grid
Ctrl + R         = Régua
Ctrl + B         = Scrollbar
Shift + Alt + P  = Paleta de cores
F11              = Tela cheia
```

## Extras
