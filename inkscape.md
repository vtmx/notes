# Inkscape

## Arquivo

```
Ctrl + N = Novo
Ctrl + Alt + N = Novo usando template
Ctrl + O = Abrir
Ctrl + S = Salvar
Ctrl + Shift + S = Salvar como
Ctrl + Shift + Alt + S = Salvar cópia
Ctrl + I = Importar
Ctrl + Shift + E = Exportar
Ctrl + P = Imprimir
Ctrl + Shift + D = Propriedades do documento
Ctrl + Q = Fechar
Ctrl + Q = Sair
```

## Edição

```
Ctrl C = Copia
Ctrl V = Cola
Ctrl + Alt + V = Cola no mesmo local
Ctrl + Shift + V = Cola estilo

Ctrl + D = Duplica
Alt + D = Clona
Alt + Shift + D = Clona sem link
Shift + D = Seleciona original

Delete = Exclui objetos selecionados
Ctrl + A = Seleciona tudo
Ctrl + Alt + A = Seleciona tudo em todas as camadas
Select same = Seleciona objetos baseado em seu estilo
! = Inverte seleção
Esc = Desceleciona
Ctrl + Shift + R = Cria canvas baseado na seleção
Ctrl + Shift + W = Cores
```

## Exibição

```
Ctrl + 5         = Modo de display
Ctrl + ;         = Guias
Ctrl + Shift + ; = Grid
Ctrl + R         = Régua
Ctrl + B         = Scrollbar
Shift + Alt + P  = Paleta de cores
F11              = Tela cheia
```

## Movimentação

```
Setas         = Move seleção
Setas + Ctrl  = Move canvas
Setas + Shift = Move seleção em incrementos maiores
Setas + Alt   = Move seleção em incrementos menores
```

## Objeto

```
# Grupo
Ctrl + G = Agrupa seleção
Ctrl + Shift + G = Desagrupa seleção

# Recorte
Ctrl + M = Adiciona clip
Alt + M  = Remove clip

# Elevação
PageUp = Eleva seleção um passo acima
PageDown = Abaixa seleção um passo a baixo
Home = Eleva seleção ao topo
End = Abaixa seleção pra baixo

# Rotação
[]        = Rotaciona seleção
Alt + []  = Rotaciona seleção em incrementos menores
Ctrl + [] = Rotaciona seleção em incrementos maiores

# Vira
H = Vira seleção horizontalmente
Shift + H = Vira seleção verticalmente

# Redimensionamento
Alt + <>  = Redimensiona seleção
Ctrl + <> = Redimensiona seleção em incrementos maiores
```

## Path

```
Ctrl + Shift + C = Converte seleção em curvas
Ctrl + Alt + C = Converte bordas em curvas
Shift + Alt + B = Converte imagem em vetor
```

## Barra de ferramentas

```
V = Seleção
A = Nodes
X = Ferramenta de forma
R = Retângulo
E = Elipse
* = Poligonal
P = Caneta
T = Texto
I = Conta gotas
Shift + P = Páginas
```

## Mouse

```
# Canvas
ClickM = Zoom in
ClickM + Shift = Zoom out
ClickD = Menu contexto

# Seleção
ClickE = Seleção de objetos
ClickE + Shift = Seleciona ou remove objeto da seleção
ClickD = Menu contexto

# Movimento
Ctrl = Movimenta confinado
Alt = Movimenta incrementando
Shift = Movimenta snapping

# Arrastando
ClickE = Seleção de objetos
ClickE + Shift = Seleciona ou remove objeto da seleção
ClickE + Alt = Linha de seleção
ClickM = Movimenta canvas

# Scroll
Scroll = Movimenta canvas verticalmente
Scroll + Shift = Movimenta canvas horizontalmente
Scroll + Ctrl = Diminui e aumenta zoom
```

## Alinhamento

```
Ctrl + Alt + {1..9} = Alinhamento estilo relógio
```

## Zoom

```
1     =  Zoom no canvas 1:1
2     =  Zoom no canvas 1:2
3     =  Zoom na seleção
4     =  Zoom englobando todos os objetos
5     =  Zoom no canvas
6     =  Zoom no canvas por largura
8     =  Zoom anterior
{=,+} = Zoom aumenta
{-,_} =  Zoom diminui
Z     =  Ferramenta de zoom
```
