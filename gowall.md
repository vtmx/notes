# gowall

Usado para converter imagens em determinadas paletas de cores.

Lista temas disponíveis:

```
gowall list
```

Exibe as cores utilizadas:

```
gowall list -t onedark
```

Converte imagem:

```
gowall convert img.jpg -t onedark --output img2.jpg
```

Tema customizado:

```
~/.config/gowall/config.yml
https://achno.github.io/gowall-docs/themes/
```
